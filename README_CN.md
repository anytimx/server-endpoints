# 服务器端点

此仓库提供了一系列备用服务器端点，当主服务端点不可用时，您可以使用它们来访问应用程序。

## 介绍

由于网络问题或服务维护等多种原因，主服务器可能无法访问。在这种情况下，为了确保我们服务的不间断访问，我们维护了一个可以使用的备用服务器端点列表。

## 如何使用

当您无法访问主服务时，请按照以下步骤使用备用服务器端点：

1. 查看下面的列表，找到一个可用的备用端点。
2. 通过更新服务器配置设置，配置您的应用程序使用新的端点。
3. 如果应用程序需要重新启动，请确保这样做以使更改生效。

## 服务器端点列表

以下是当前可用的备用服务器端点列表：

- `https://anytimx.com`

请注意，这些端点的可用性可能会发生变化，并且可能会定期更新。
